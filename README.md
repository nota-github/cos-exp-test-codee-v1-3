# 오늘의 미니 체크인

사용자가 오늘의 기분, 할 일 3개, 짧은 메모를 한 화면에서 기록할 수 있는 아주 간단한 웹 앱을 만든다. 저장 기능은 브라우저 로컬 저장소만 사용한다.

## 현재 구현 범위

현재 워크스페이스는 `story-1.2` 기준으로 상단 헤더의 device-local 오늘 날짜 표시까지 포함한다. 기분 선택 상태, 할 일 상태, 메모 상태, localStorage 복원은 후속 스토리에서 연결된다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 표시된 로컬 주소를 열면 상단 헤더에 오늘 날짜가 먼저 보이고, 그 아래에 기분, 할 일, 메모 입력 영역이 같은 화면 흐름으로 이어지는 대시보드를 확인할 수 있다.

정적 빌드는 아래 명령으로 확인한다.

```bash
npm run build
```

스토리 루프 자동화 검증에서는 watch mode 대신 아래 종료형 명령을 사용한다.

```bash
npm run validate:story-1.1
npm run validate:story-1.2
```
