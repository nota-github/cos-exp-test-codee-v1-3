# Implementation Summary

- `src/features/todos/todoSlots.ts`를 추가해 fixed three-slot todo UI를 전용 모듈로 분리했다. `TODO_SLOTS` 상수에 정확히 3개의 row 정의만 두고, 각 row에서 `${slot.id}-checkbox`, `${slot.id}-input` 고유 id를 만들어 checkbox/text input을 독립 연결했다.
- `src/app/createApp.ts`에서 기존 인라인 `todoExamples` 기반 마크업을 제거하고 `createTodoSlots('todo-title')`로 교체했다. 이에 따라 `story-1.4:SCOPE-1`, `story-1.4:SCOPE-2`, `story-1.4:AC-1`, `story-1.4:AC-2`가 전용 모듈을 통해 반영됐다.
- `src/styles.css`에서 `.todo-row`를 고정 2열 grid로 정리하고, checkbox container에 별도 hit area, border, focus-within state, 모바일 간격 조정을 추가했다. 이 변경으로 `story-1.4:SCOPE-4`와 `story-1.4:AC-5`의 checkbox/input 간섭 완화 요구를 story 범위 안에서 다뤘다.
- `scripts/validate-story-1.4.mjs`와 `package.json`의 `validate:story-1.4`를 추가해 `typecheck -> build -> story-1.4 구조/스타일/범위 검사`가 종료 코드와 함께 끝나도록 만들었다.
- 저장/복원과 완료 항목의 후속 표현은 이번 story 범위 밖이라 건드리지 않았다.
